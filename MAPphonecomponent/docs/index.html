Client-side installation

 

This documentation presumes that you already have DHTMLX Suite PRO version and CAIRS Framework installed on your system.

 

1 - To install, place the following Javascript and CSS files at the <head></head> section of your page

		<!-- DHTMLX -->
<link rel="stylesheet" type="text/css" href="../../auxiliary/dhtmlxfull3.5/dhtmlx.css" />
<script type="text/javascript" src="../../auxiliary/dhtmlxfull3.5/dhtmlx.js" charset="ISO-8859-1"></script>
		<script type="text/javascript" src="js/json2.js"></script>      
         
        <script type="text/javascript" src="model/phonecomponent_Model.js"></script>
        <script type="text/javascript" src="controller/phonecomponent.js"></script>

Now the Phone component is already installed on your system.

To use the component just call the phonecomponent.start() method inside your application.

Complete call


				  phonecomponent.start( { 
                        uid : "86081" 
                        ,application_path : "http://test.myadoptionportal.com/modules/MAPphonecomponent/" 
                        ,dhtmlx_codebase_path : "http://test.myadoptionportal.com/auxiliary/dhtmlxfull3.5/" 
						,contactid : "86081"
						,agencyid: 25
						,useWindow : true
     					,parentDIVId : ''
                    } );
